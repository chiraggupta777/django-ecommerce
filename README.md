# Django E-commerce Project

## Features
- User Signup/Login
- Product Listing
- Search functionality
- Cart system
- Checkout system

## Tech Stack
- Django
- HTML/CSS
- SQLite

## How to Run
1. Clone the repo
2. Create virtual environment
3. Install requirements
4. Run: python manage.py runserver
5. ## Project Flow
- User registers or logs in
- Products are fetched from database using Django models
- User adds items to cart (session-based storage)
- Cart data is processed during checkout
- Order details are stored in database

## Key Concepts Used
- Django ORM for database handling
- Session management for cart
- Template rendering for dynamic UI
